Description of the webpage

In this webpage you can navigate between the options shown on top of the page. This is the Home page with a small description of the show in which we based our website.

in the List section you will see the characters of the show giving some aditional information of each one by clicking on the button 'more info'. On top of that you can also filter and search for your favorite character.

In the Location section you will find also a list but this time is a list of the locations in which the show has been set. You can create new planets edit their names or delete them from the universe, you are the boss.

Also if you get lost dont worry we got you covered ;), there is a not-found warp that will take you to a safe zone called the 404 dimention :D
